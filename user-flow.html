User Flows

Accessible pathways for primary personas

Margaret Thompson

Age 72 Low Vision Tablet User Keyboard Navigation

Retired teacher who needs large text and clear pathways to find library services.

Flow 1: Finding Library Hours

Margaret needs to check when the main branch is open before visiting.

1

Homepage Load

Margaret lands on the homepage and immediately activates the text zoom to 150%.

♿ Text reflows cleanly without horizontal scrolling. Skip link available to bypass header.
2

Access Hours Information

She looks for the hours information in the main navigation.

♿ Mega menu appears on focus with large clickable areas (48x48px). Screen reader announces "Hours and Locations section".
3

View Branch Hours

She selects "Main Branch" from the locations list.

♿ Accordion expands with high-contrast text. Keyboard users can tab through all branches.

Flow 2: Planning an Accessible Visit

Margaret wants to confirm accessibility features before her visit.

1

Navigate to Accessibility Hub

She clicks the "Accessibility" link in the primary navigation.

♿ Persistent accessibility toolbar remains visible. Link has 3:1 contrast ratio.
2

Explore Interactive Map

She uses the building map to locate elevators and quiet zones.

♿ Map has text alternatives and can be navigated via keyboard. Landmarks announced by screen reader.
3

Request Accommodations

She completes the "Plan Your Visit" form to request a magnifier.

♿ Form fields have visible labels and error messages. Timeout warning for slow typists.

Flow 3: Attending an Event

Margaret wants to register for a senior book club meeting.

1

Find Events Calendar

She uses the quick-access "Events" card on the homepage.

♿ Card has aria-label="Events calendar" and large hit area (minimum 44x44px).
2

Filter for Accessible Events

She applies the "Large Print" and "Wheelchair Accessible" filters.

♿ Filter controls are keyboard operable. Selected filters announced by screen reader.
3

Complete Registration

She registers using the simplified form with her existing library account.

♿ Form autocompletes known information. Success message includes confirmation email details.
