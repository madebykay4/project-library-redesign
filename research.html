Evidence-based findings to inform an accessible, user-centered redesign

Research Methodology

1. User Surveys & Interviews

Participants: 32 library patrons (18 seniors, 8 visually impaired users, 6 caregivers)

Key Questions:

  • What tasks do you struggle to complete on our website?
  • How do you typically adjust text size or contrast?
  • Where would you expect to find accessibility information?

2. Accessibility Audits

Tools Used: WAVE, Axe, ANDI

Scope: Full WCAG 2.1 AA evaluation of 12 key pages

Major Findings:

  • 38 instances of insufficient color contrast
  • 24 missing form labels
  • 15 keyboard navigation traps

3. Usability Testing

Tasks:

  • Find library hours for the main branch
  • Locate information about wheelchair accessibility
  • Register for an event using screen reader

Key Findings

1. Text Accessibility Barriers

  • Default text size too small for 85% of seniors surveyed
  • No persistent text resizing controls available
  • Text overflow when zoomed to 150%+
"The text is too small, and I can't figure out how to make it bigger."
— Participant #07, age 74

2. Navigation Challenges

  • Average 3.4 minutes to locate accessibility information
  • Dropdown menus inaccessible to keyboard/screen reader users
  • 67% task failure rate for event registration

3. Missing Accessibility Information

  • No centralized accessibility resource page
  • Building maps lack accessible route markers
  • Screen reader users couldn't locate operating hours

Design Recommendations

1. Implement WCAG 2.1 AA Standards

  • Add persistent accessibility toolbar (text resize, contrast toggle)
  • Ensure proper heading structure and ARIA landmarks
  • Fix all color contrast issues (minimum 4.5:1 ratio)

2. Simplify Navigation

  • Replace dropdowns with mega menu featuring large clickable areas
  • Add predictive search with autocomplete
  • Create clear "Accessibility Hub" in primary navigation

3. Add Visit Planning Tools

  • Interactive accessibility map showing elevators, quiet zones
  • Accommodation request form (ASL interpreters, etc.)
  • Service animal policy prominently displayed
