User Personas

Key user groups for the library redesign

Margaret Thompson

Retired Teacher with Low Vision

Age 72 Low Vision Tablet User Keyboard Navigation

Retired educator who relies on the library for large print books and community events. Needs simple navigation and adjustable text sizes.

"I just want to find the library hours without asking my grandson for help every time."

Goals

  • Adjust text size without horizontal scrolling
  • Find accessibility features before visiting
  • Navigate using keyboard only
  • Learn about senior-focused programs

Frustrations

  • Small clickable areas that are hard to target
  • Complex multi-level navigation menus
  • Lack of clear accessibility information
  • Forms that timeout before she can complete them

James Wilson

College Student with Dyslexia

Age 19 Dyslexia Student Mobile First

Community college student who uses the library's study spaces and digital resources. Needs dyslexia-friendly formatting options.

"If I can't change the font, I can't use the website. It's that simple."

Goals

  • Switch to OpenDyslexic font
  • Find study room reservation system
  • Access digital resources from phone
  • Quickly check computer lab availability

Frustrations
  • Dense blocks of justified text
  • Complex academic database interfaces
  • No text-to-speech for articles
  • Mobile-unfriendly forms

Maria Garcia

ESL Parent with Limited Tech Experience

Age 34 Spanish Speaker Mobile Only Language Barrier

Mother of two who visits the library for children's programs and ESL resources. Needs simple language options and visual navigation.

"I know the library has great kids programs, but the website is too complicated in any language."

Goals

  • Find Spanish-language resources
  • See pictures of library spaces
  • Register children for storytime
  • Get directions via public transit

Frustrations

  • Complex library jargon
  • Text-heavy pages with few visuals
  • Small touch targets on mobile
  • No translation toggle

Common Accessibility Needs

Visual

  • Text resizing to 200%
  • High contrast mode
  • Screen reader compatibility
  • Alternative text for images

Motor

  • Keyboard navigation
  • Large clickable areas
  • Voice control support
  • No time-limited tasks

Cognitive

  • Simple language options
  • Consistent navigation
  • Dyslexia-friendly fonts
  • Clear error messages
